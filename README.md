# arkiver

a wrapper for multiple archival utilities to list and extract archives from
tarballs, zip, 7zip, rar, and more.


|program|archive|
|-------|-------|
|tar|tarballs|
|xz utils|xz and lzma|
|bzip2|bz2|
|unrar|unzip|
|gzip|gz|
|7z|7zip and zip|
|uncompress|Z archives|
|zstd|zst|
|ar|deb|
|cabextract|cab and exe|
|unar|old mac and amiga formats|


## installation

```sh
make install
```
